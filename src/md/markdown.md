# 마크다운 언어란 무엇인가?

마크다운 언어는 텍스트 기반의 마크업 언어로 HTML처럼 태그 없이도 글과 특수기호만으로도 문서 구조를 빠르게 구성할 수 있는 언어이다.

# 문법

## 제목

- # 제목1 : #쓰고 한칸 띄우면 제목이다
- ## 제목2
- ### 제목3
- #### 제목4

## 글자강조

**굵게**

_기울임(Italic)_

~~취소선~~

**_~~전부 쓰는 것도 가능~~_**

## 목록

1. 안녕하세요
2. 숫자. 쓰고 한칸 띄면
3. 숫자 리스트가 만들어지고요

- +나-를 이용하면
- 순서없는 정렬도 가능
  - 들여쓰기 가능^^

## 코드 넣기

`HELLO WORLD`
이런식으로 백틱을 활용하면 코드도 넣을 수 있음
>HELL~~O WORLD~~

## 체크박스도 가능

- [ ] 개발 공부를 하였나요?
- [x] 오늘 놀았나요?
- [x] 방에서 도파민을 수집했나요?
- [x] 돼지마냥 굴러다녔나요?
- [x] 방에 거지가 있나요?
